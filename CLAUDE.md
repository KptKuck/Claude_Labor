# Claude Code Projekteinstellungen

## Projekt
- **Name:** BTCUSD Analyzer
- **Sprache:** MATLAB
- **Zweck:** BILSTM Neural Network für BTC Trendwechsel-Erkennung

## Kommunikation
- **Sprache:** Deutsch
- **Commit-Messages:** Englisch
- **Kommentare im Code:** Deutsch

## Code-Style
- MATLAB App Designer / uifigure für GUIs
- Funktionen mit ausführlichen Docstrings
- Variablennamen: snake_case (z.B. `training_data`, `gpu_status_label`)
- Konsistente Einrückung: 4 Spaces

## Git
- Commit-Messages auf Englisch
- Co-Authored-By Tag bei Commits
- Push nur auf explizite Anfrage

## GUI-Design
- Dunkles Theme (BackgroundColor: [0.15, 0.15, 0.15])
- Farbschema:
  - Erfolg/OK: [0.2, 0.7, 0.3] grün
  - Warnung: [0.9, 0.7, 0.2] orange
  - Fehler/Stopp: [0.8, 0.3, 0.2] rot
  - Neutral: [0.5, 0.5, 0.5] grau
- Buttons mit weißer Schrift, fett
- 'fit' für dynamische Layouts bevorzugen

## Präferenzen
- Keine Emojis im Code oder Kommentaren
- Kompakte Antworten bevorzugt
- Bei Änderungen: Nur relevante Dateien modifizieren
- Neue Dateien nur wenn nötig erstellen
